# Customer_behavior_analysis
Data Analytics Project showcashing customer behaviour analysis using Python, SQl and PowerBI.
The project analyzes 3,900 purchases to understand spending, segments, and subscription behavior.
Young Adults contribute the highest revenue, requiring targeted marketing focus.
Top-rated products include Gloves, Sandals, and Boots, indicating high customer satisfaction.
Only 27% are subscribers, but repeat buyers are much more likely to subscribe.
Strategic recommendations focus on boosting subscriptions and rewarding the Loyal customer segment.
